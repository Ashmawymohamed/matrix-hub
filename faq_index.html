FAQ

How do I load an effect?
Download the .cfg file, copy it to the /effects folder on your SD card, then use the SD panel in the main page.
What are the effect IDs?
Each .cfg file contains an ID= line. This number determines which effect runs on the matrix.
Can I create my own effects?
Yes! Create a .cfg file with ID, SPEED, and BRIGHT lines, and save it to /effects on your SD card.
